# Mini Insta

A lightweight Instagram-inspired full-stack app built with Node.js, Express, and a vanilla JavaScript frontend.

## Features

### 2. User Profile

- Profile Picture
- Bio
- Followers Count
- Following Count

### 3. Create Posts

- Text Posts
- Image Upload
- Timestamp

### 4. Like & Comment System

- Like/Unlike Posts
- Add Comments
- Delete Own Comments

### 5. Follow/Unfollow Users

- Follow Other Users
- View Followers and Following Lists

### 12. Dark Mode

Switch between Light and Dark themes.


### 14. Search Users

- Search by Username
- View Profiles

### 15. Infinite Scroll

Load posts automatically as users scroll.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Open the frontend in the browser and start exploring the social feed.
